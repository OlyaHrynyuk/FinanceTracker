# Веб-додаток для відстеження особистих фінансів

## Опис проєкту
Personal Finance Tracker — це веб-додаток, який допомагає користувачам контролювати власні доходи та витрати.  
Користувач може додавати записи про фінансові операції, переглядати історію транзакцій і бачити поточний баланс.

Проєкт створено в рамках навчальної дисципліни «Створення ІТ-проєктів» з метою опанування процесу розробки MVP, роботи з Git та публікації на GitHub.

---

## Основні можливості (MVP)
- Додавання доходів і витрат вручну.  
- Перегляд загального балансу.  
- Відображення історії транзакцій.  
- Збереження даних у LocalStorage (зберігаються після перезавантаження сторінки).  
- Адаптивний інтерфейс для комп’ютера та мобільних пристроїв.

---

## Використані технології
- HTML5 — структура сторінок  
- CSS3 — стилізація інтерфейсу  
- JavaScript (ES6) — логіка взаємодії з даними  
- Git / GitHub — контроль версій та хостинг  
- (За потреби) Figma — створення прототипу інтерфейсу
