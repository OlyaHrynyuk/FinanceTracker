# FinanceTracker

## Опис проєкту
Веб-додаток для відстеження особистих фінансів. Користувач може додавати доходи, витрати та переглядати їх статистику.

## Мета
Допомогти користувачу контролювати фінанси та аналізувати свої витрати.

## Використані технології
- HTML
- CSS
- JavaScript

## Інструкція запуску
1. Завантажити код з GitHub.
2. Відкрити файл index.html у браузері.
