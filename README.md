#  FinanceTracker

## Опис Проєкту
Цей застосунок є Minimum Viable Product (MVP) для відстеження особистих фінансів. Він дозволяє користувачам додавати транзакції (доходи та витрати), автоматично розраховувати поточний баланс та відображати історію фінансових операцій. Проєкт реалізовано як односторінковий веб-додаток, демонструючи навички роботи з DOM-маніпуляціями та обробкою даних у чистому JavaScript.

##  Мета Проєкту
* Створення функціонального MVP для управління особистими фінансами.
* Закріплення навичок Frontend-розробки (HTML/CSS для інтерфейсу).
* Реалізація логіки обробки форм та динамічного оновлення інтерфейсу за допомогою JavaScript.

##  Використані Технології
* HTML5
* CSS3
* JavaScript (ES6+) – основна логіка.
* Git / GitHub – контроль версій.

##  Основний Функціонал (MVP)
* Додавання транзакцій: Форма для введення опису та суми. Дохід визначається як сума > 0, витрата – як сума < 0.
* Розрахунок Балансу: Динамічне обчислення загального поточного балансу за допомогою методу Array.prototype.reduce().
* Відображення Історії: Список всіх доданих операцій з відображенням опису та суми.
* Валідація: Базова перевірка коректності введених даних.

##  Інструкція Запуску

Щоб запустити проєкт локально:

1.  Клонуйте репозиторій:
       git clone [https://github.com/OlyaHmyuk/FinanceTracker.git](https://github.com/OlyaHmyuk/FinanceTracker.git)
    2.  Перейдіть у папку проєкту:
       cd FinanceTracker
    3.  Відкрийте файл `index.html` у будь-якому сучасному веб-браузері (наприклад, Chrome або Firefox).
